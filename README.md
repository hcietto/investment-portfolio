# Investment Portfolio Data Pipeline

This project is a simple data engineering pipeline that collects stock prices from a public API, processes the data, and stores it in an Excel file for portfolio analysis.

## Technologies
- Python
- Pandas
- Yahoo Finance API
- Excel

## Pipeline Steps
1. Extract stock prices from API
2. Transform and clean the data
3. Load data into Excel

## How to Run
```bash
pip install -r requirements.txt
python src/main.py
